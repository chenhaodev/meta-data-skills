## 介绍

MedCOT数据集包含需要多步推理的复杂临床案例，覆盖门诊、急诊和慢性病管理等三大类场景，每个案例提供症状表现、检查结果和病史信息。模型需要展示从症状识别到诊断结论的完整推理过程，包括鉴别诊断、证据权重分析和诊断依据等环节。数据集旨在评估模型推理链条的逻辑性、临床准确性和决策透明度。

MedBench评测榜单：该任务的评测采用评价模型LLM-as-a-Judge作为评估指标。

## 元数据

数据集中包含以下信息

```
<div><p>question：复杂临床案例
answer：完整推理过程
</p></div>
```

## 示例

```
<div><p><span>{</span>
    <span>"question"</span><span>:</span> <span>"症状表现： 患者，男，26岁，大学生。近5天出现发热、乏力及进行性头痛；2天前出现恶心、呕吐并逐渐出现意识混乱及短暂言语不清，今晨出现一次全身阵挛性抽搐（持续约1分钟，随后意识未完全清醒）。家属报告数小时内发现患者对周围刺激反应迟钝。既往体健，无慢性疾病及长期用药史。否认外伤史或近期旅行史。

体格与神经检查： 体温 39.2℃，脉搏 100次/分，呼吸 18次/分，血压 130/80 mmHg。神志淡漠，对时间地点识别不清；可见右侧面部轻度周围性面瘫（眉毛可动，口角下垂），右上肢肌力4/5，颈强直阳性；瞳孔等大等圆，对光反射存在。无皮疹或出血点。

辅助检查： 血常规：WBC 12.2×10^9/L（中性粒细胞78%）；CRP 28 mg/L（升高）。头颅CT（急查）未见占位性病变或大出血。腰椎穿刺：开颅压约 220 mmH2O（轻-中度升高），脑脊液（CSF）外观微浑浊，细胞计数 180 cells/µL，以淋巴细胞为主（约70%）；蛋白 1.2 g/L（显著升高），葡萄糖 2.6 mmol/L（血糖同日测 6.0 mmol/L，CSF/血糖比率 0.43）；CSF革兰染色未见细菌，常规培养待出结果。脑MRI（扩展检查）显示右侧颞叶及颞极 FLAIR 高信号，伴轻度皮层坏死征象；脑电图（EEG）：右侧颞区周期性放电性样改变（PLDs）。血培养 48 小时内阴性（仍在观察）。

流行病学/暴露史： 无明确暴露史，否认吸毒及高危性行为，无家族类似病史。"</span><span>,</span>
    <span>"answer"</span><span>:</span> <span>"步骤1 (识别关键特征): 
    
    关键临床特征组合为：
    
    1.亚急性起病的发热、乏力等全身症状；
    
    2.与抓伤部位同侧的局部、痛性淋巴结肿大（左腋窝）；
    
    3.抓伤部位存在原发皮损（左手食指丘疹）
    
    4.明确的动物（猫）接触史和抓伤史；
    
    5.实验室检查提示炎症反应（WBC、CRP升高）但血培养阴性。
    
    步骤2 (鉴别诊断): 
    
    基于“发热伴局部淋巴结肿大”这一核心表现，需考虑以下鉴别诊断：
    
    1.猫抓病（Bartonella henselae感染）：流行病学史（猫抓伤）、原发皮损和局部淋巴结炎的三联征高度吻合。
    
    2.细菌性淋巴结炎（如金黄色葡萄球菌、链球菌感染）：也表现为发热和痛性淋巴结肿大，通常由明显的皮肤软组织感染引起。
    
    3.传染性单核细胞增多症（EB病毒感染）：常表现为发热、咽痛和淋巴结肿大，但淋巴结肿大通常是全身性的，尤以颈部为著。
    
    4.淋巴瘤：作为恶性疾病，可引起发热（B症状）和淋巴结肿大，但淋巴结通常为无痛性、进行性肿大。
    
    5.弓形虫病：有动物接触史，可引起发热和淋巴结肿大，但淋巴结通常无痛。
    
    步骤3 (证据权重分析):
    
    1.支持猫抓病的证据权重最高：患者有明确的猫抓伤史，且抓伤部位（左手）与疼痛性淋巴结肿大区域（左腋窝）在引流关系上完全吻合。原发皮损、淋巴结炎和全身症状的出现顺序和时间间隔（约2周）也与猫抓病的典型病程一致。血培养阴性也符合巴尔通体为苛养菌、常规培养难以生长的特点。
    
    2.反对细菌性淋巴结炎的证据：原发皮损并非典型的化脓性感染灶（如脓疱或蜂窝织炎），而是一个丘疹。若是常见细菌感染，病情发展可能更急骤，或局部化脓更明显。
    
    3.反对传染性单核细胞增多症的证据：患者无明显咽痛，且淋巴结肿大局限于单侧腋窝，而非全身性或以颈部为主。
    
    4.反对淋巴瘤的证据：淋巴结有明显压痛，提示急性炎症过程，而非肿瘤的典型特征（无痛、质韧）。患者年龄较轻，且有明确的感染前驱事件。
    
    5.反对弓形虫病的证据：弓形虫病淋巴结通常无痛，且传播途径多为消化道，与抓伤的直接关系不大。
    
    步骤4 (诊断依据): 
    
    综合分析，患者的流行病学史、典型的临床三联征（原发损、局部淋巴结炎、发热）以及实验室检查结果，高度指向猫抓病的诊断。其诊断逻辑链条为：猫抓伤（病因暴露） -&gt; 局部皮肤出现原发接种丘疹 -&gt; 病原体沿淋巴管引流 -&gt; 到达所属区域淋巴结（左腋窝）并引发炎症反应 -&gt; 病原体入血引起菌血症及全身非特异性症状（发热、乏力、皮疹）。
   
    步骤5 (行动建议与治疗计划):
    
    1.诊断确认：建议行巴尔通体血清学抗体检测（Bartonella henselae IgM和IgG），IgM阳性或双份血清IgG滴度4倍以上升高可确诊。如诊断困难或淋巴结持续肿大，可考虑行淋巴结穿刺活检。
    
    2.治疗方案：虽然猫抓病对免疫功能正常的个体有自限性，但该患者症状较重（高热、明显疼痛），建议给予抗生素治疗以缩短病程、缓解症状。首选药物为阿奇霉素，口服，疗程5-7天。
    
    3.对症支持：使用非甾体抗炎药（如布洛芬）以退热和缓解淋巴结疼痛。
    
    4.随访与宣教：告知患者疾病大多预后良好，注意休息。嘱其监测体温和淋巴结变化，如症状加重或出现新症状（如视力模糊、肝脾肿大等）应及时复诊。进行宠物接触安全教育。"</span>
<span>}</span>
</p></div>
```
