## 介绍

MedCollab数据集包含需要多系统协作的复杂医疗场景，如诊断辅助类协作、治疗执行类协作、慢病管理类协作、应急协同类协作、康复指导类协作五种协作方式等。每个场景提供分布式系统环境和协作需求描述。模型需要协调多个工具和应用完成复杂任务。数据集旨在评估模型的任务分解能力、系统协调性和工作流优化能力。

MedBench评测榜单：该任务的评测采用评价模型LLM-as-a-Judge作为评估指标。

## 元数据

数据集中包含以下信息

```
<div><p>question：医保场景信息和核心目标
answer：规划协作序列结果
</p></div>
```
