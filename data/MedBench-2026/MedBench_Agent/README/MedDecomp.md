## 介绍

MedDecomp数据集包含丰富的临床场景描述和核心医疗目标，如门诊诊疗、急诊急救、慢病管理、术后康复、公共卫生等5大类。每个案例均提供患者背景、当前健康状况和最终诊疗目标。模型需要将抽象目标分解为可执行的具体任务序列，包括检查安排、治疗措施、健康监测等环节。该任务旨在评估模型的任务分解的逻辑性、临床可行性和流程全面性。

MedBench评测榜单：该任务的评测采用评价模型LLM-as-a-Judge作为评估指标。

## 元数据

数据集中包含以下信息

```
<div><p>question：场景描述和医疗目标
answer：任务序列
</p></div>
```

## 示例

```
<div><p><span>{</span>
    <span>"question"</span><span>:</span> <span>"场景背景：

一名 55 岁男性，公司职员，因“糖尿病随访”到内分泌门诊复查。患者于 3 年前体检时发现空腹血糖升高，经进一步检查确诊为“2 型糖尿病”。目前口服二甲双胍 0.5g 每日两次，规律服药。自述近 2 周出现轻度乏力、口干，夜尿次数较以往增加至 2～3 次/夜。无明显视物模糊或下肢麻木。最近体重较上月增加约 1.5kg。自测血糖多在空腹 8.2～9.0 mmol/L。日常饮食控制一般，偶有应酬。既往有高血压病史 5 年，规律服用缬沙坦 80mg 每日一次，血压控制尚可。

核心目标：

评估该患者 2 型糖尿病控制情况及相关并发症风险，分析可能的血糖升高原因，优化个体化降糖方案并强化生活方式干预；通过系统的检查、教育与随访，帮助患者实现血糖达标与并发症预防。"</span><span>,</span>
    <span>"answer"</span><span>:</span> <span>"一、病史采集与初步评估（执行主体：门诊医生）
    
 子任务1：医生引导患者就坐，询问近 1 个月的血糖监测记录，包括空腹血糖、餐后 2 小时血糖及测量时间规律性；

 子任务2：详细询问患者近期饮食习惯变化（碳水摄入量、外出就餐频率、饮酒情况）及体重变化趋势；

 子任务3：了解用药依从性，确认是否按时服药、是否存在漏服、服药后不适等问题；

 子任务4：评估运动习惯：每周运动次数、持续时间及运动形式；

 子任务5：询问低血糖史（如手抖、出汗、心悸发作时间及处理方式）；

 子任务6：了解并发症症状：视物模糊、手脚麻木、下肢水肿、夜尿增加等；
 
 子任务7：复测血压，记录生命体征并计算BMI，初步评估代谢综合征风险。


二、体格检查与实验室检查安排（执行主体：门诊医生）

 子任务8：进行双足检查：观察皮肤完整性、足背动脉搏动、感觉功能（针刺或音叉试验）；

 子任务9：开具实验室检查项目：

空腹血糖与糖化血红蛋白（HbA1c）

空腹血脂谱（TC、TG、LDL-C、HDL-C）

肾功能（血肌酐、尿微量白蛋白）

肝功能（ALT、AST）

尿常规

 子任务10：必要时开具心电图，评估潜在的心血管风险。

三、病情分析与诊疗决策（执行主体：门诊医生）

 子任务11：调阅实验室检查结果：若HbA1c &gt;7.0%，说明血糖控制不达标；结合空腹血糖与体重变化，推测可能与饮食控制不严及体重上升相关；

 子任务12：评估肾功能：若肾功能正常，可继续使用二甲双胍；

 子任务13：若患者BMI&gt;28 kg/m²，可考虑联合SGLT2抑制剂或GLP-1受体激动剂；
 
 子任务14：若患者空腹血糖持续≥8 mmol/L，且HbA1c&gt;7.5%，可考虑加用DPP-4抑制剂；

 子任务15：若患者伴血压轻度升高或微量白蛋白尿，强化ACEI/ARB治疗。


四、健康教育与生活方式指导（执行主体：门诊医生+糖尿病教育护士）
 
 子任务16：教育护士向患者讲解“平衡膳食模式”：每日主食控制在200-250g，增加蔬菜摄入，减少油炸及含糖饮料；

 子任务17：强调规律运动：建议每周至少150分钟中等强度有氧运动（如快走、骑车），并进行阻力训练；

 子任务18：指导家庭血糖监测：建议每日早晚测血糖各一次，记录血糖曲线并带回下次复诊；

 子任务19：讲解低血糖识别与处理：如出现手抖、出汗应立即进食含糖食物；

 子任务20：强调戒烟限酒的重要性，告知其长期吸烟会加重血管损伤。

五、随访与评估计划（执行主体：门诊医生）

 子任务21：制定随访周期：血糖未达标者每月随访一次，稳定后每3个月复查；

 子任务22：每次随访需评估体重、血压、血糖及用药依从性；

 子任务23：每年进行一次糖化血红蛋白、尿微量白蛋白及眼底检查；

 子任务24：若后续血糖仍控制不佳，评估是否需胰岛素治疗；

 子任务25：建立电子健康档案，记录随访数据以便长期管理。"</span>
<span>}</span>
</p></div>
```
