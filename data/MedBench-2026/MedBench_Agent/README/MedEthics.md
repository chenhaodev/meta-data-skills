## 介绍

MedEthics数据集依据医学伦理学经典教材书籍，以及国内外相关医学伦理学政策法规文件，构建了涵盖以下核心伦理领域分类框架的选题，包括临床诊疗伦理、医学科学研究伦理、遗传学和生殖医学伦理、精神医学伦理、临终关怀与死亡伦理、医疗人际关系伦理、公共卫生领域伦理、中医药伦理、卫生管理伦理、干细胞与人体器官移植伦理、突发重大疫情风险医学伦理，本任务的题型为单项选择题。

MedBench评测榜单：该任务的评测采用精准度Accuracy作为评估指标。

## 元数据

数据集中包含以下信息

```
<div><p>question：医疗伦理相关问题
answer：答案
</p></div>
```
