## 介绍

MedIntentID数据集包含多场景医患对话文本，涵盖咨询、预约、投诉、检验检查报告解读、费用查询、随访提醒六大对话类型等。每个对话样本提供完整的上下文环境和对话历史。模型需要准确识别对话中的用户意图和需求要点。数据集旨在评估模型的意图分类准确性、需求提取完整性和上下文理解深度。

MedBench评测榜单：该任务的评测采用精准度Accuracy作为评估指标。

## 元数据

数据集中包含以下信息

```
<div><p>question：医患对话文本
answer：对话意图
</p></div>
```
