## 介绍

MedReflect 数据集涵盖门诊、住院初步诊疗、慢病管理、术后康复四大场景，每个案例均提供主诉、现病史、辅助检查、诊断和初始诊疗方案，模型需识别案例决策中的偏差和错误，提出改进方案并阐明修正依据。该数据集旨在评估模型的错误识别准确性、修正方案合理性及反思深度。

MedBench评测榜单：该任务的评测采用评价模型LLM-as-a-Judge作为评估指标。

<div><p><span>{</span>
    <span>"question"</span><span>:</span> <span>"主诉：血糖控制不佳，复查调整用药。

现病史：患者，男，56岁，5年前确诊2型糖尿病，平时口服二甲双胍 0.5g 每日2次。近1个月因工作繁忙及饮食不规律，晨起空腹血糖多在 9~10 mmol/L，餐后2小时血糖 13~15 mmol/L，自觉口干、多尿，无体重明显变化。无视物模糊、下肢水肿、胸闷或呼吸困难。既往无冠心病、脑梗或肾病史。

体格检查：身高170 cm，体重82 kg，BMI 28.4 kg/m²。血压 148/92 mmHg。

实验室检查：

HbA1c：8.2%；肝肾功能正常；血脂：TC 5.9 mmol/L，TG 2.3 mmol/L。

初始诊疗方案：

诊断：2型糖尿病，血糖控制不佳。

治疗方案：继续口服二甲双胍 0.5g 每日2次，建议加强饮食控制和运动锻炼；1个月后复查血糖。
随访医嘱：每日监测空腹血糖并记录，如血糖持续高于10 mmol/L再来复诊。"</span><span>,</span>
    <span>"answer"</span><span>:</span> <span>"错误类型： 慢病管理场景 – 治疗方案调整不足/血糖监测与并发症风险评估缺失。

具体描述：

医生的治疗方案存在明显不足。面对糖化血红蛋白（HbA1c 8.2%）提示的持续血糖控制不良，仅维持原有二甲双胍单药治疗，而未进行药物升级或合并治疗调整，缺乏循证依据。其次，随访计划过于被动，仅建议“如血糖高再复诊”，容易延误病情控制。该方案未评估患者肥胖（BMI 28.4 kg/m²）和高血压（148/92 mmHg）等心血管危险因素，也未提出针对性干预（如降压或调脂治疗）。此外，患者的自我监测频次和内容过于局限，未包含餐后血糖、夜间低血糖监测或长期并发症筛查安排。整体方案反映出慢病管理连续性和主动干预意识的缺乏。


修正要求：

识别错误：初始方案未依据血糖控制水平及时优化降糖治疗方案，忽视了合并危险因素的系统性干预与随访安排，缺乏分层管理思维。

提出修正方案：

药物治疗优化：

依据《2型糖尿病防治指南（2024版）》：HbA1c ≥7.5% 且已使用二甲双胍单药者，应考虑联合用药。

建议在原方案基础上加用第二种降糖药，如：DPP-4抑制剂（如西格列汀 100mg qd）或SGLT2抑制剂（如达格列净 10mg qd），以改善血糖控制并兼顾心肾保护效益。

评估患者是否适合GLP-1受体激动剂（如利拉鲁肽），尤其在超重和心血管风险升高者中。
并发症风险综合管理：

评估并控制血压：目标 &lt;130/80 mmHg，可考虑启动ACEI或ARB类药物。

调整血脂：TG、TC升高提示动脉粥样硬化风险，应启动他汀类药物（如阿托伐他汀 20mg qn）。

建议进行基础并发症筛查：眼底检查、尿微量白蛋白、足部神经功能检查。

随访与监测改进：

自我监测血糖频次应增加至每日空腹+餐后2小时血糖；必要时增加睡前监测，记录波动趋势。

1个月内安排随访，重点评估新方案耐受性与血糖变化。

每3个月复查 HbA1c、肝肾功能、血脂，动态调整用药。

生活方式与患者教育：

建立个体化营养与运动计划，明确目标体重（BMI&lt;25 kg/m²）。

重点宣教低血糖识别与应对措施。

推荐加入糖尿病管理微信群或社区慢病管理项目，确保信息沟通与依从性。

阐述修正依据：

循证治疗原则：依据国内外指南（如ADA 2024、中华医学会糖尿病学分会指南），当单药治疗无法控制血糖时，需及时联合治疗，以防糖尿病慢性并发症进展。

风险整合管理：糖尿病与高血压、高血脂常并存，是心血管事件的主要危险因素。综合干预能显著降低卒中和心梗风险。

监测科学性：动态血糖监测能反映波动性和药物效果，是优化治疗的关键依据。仅靠空腹血糖无法反映全面控制情况。

患者安全与依从性：定期随访与主动调整是慢病管理的核心环节。教育与管理结合可显著提高治疗依从性与长期控制率。"</span>
