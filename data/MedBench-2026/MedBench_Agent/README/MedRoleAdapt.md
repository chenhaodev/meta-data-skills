## 介绍

MedRoleAdapt数据集包含多角色医疗场景对话，涵盖患者、家属、临床医生、技师、药师、康复师、护士等不同角色。每个场景提供角色信息、对话背景和沟通目标。模型需要根据角色特征调整回复内容和表达方式。数据集旨在评估角色特征把握、沟通方式适配合适性和专业度控制。

MedBench评测榜单：该任务的评测采用评价模型LLM-as-a-Judge作为评估指标。

## 元数据

数据集中包含以下信息

```
<div><p>question：角色和对话需求
answer：医疗场景对话
</p></div>
```
